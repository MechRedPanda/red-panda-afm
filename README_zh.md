# red-panda-afm
低成本DIY原子力显微镜！

## 背景
本项目旨在使用3D打印机创建低成本的原子力显微镜（AFM）。目标是让AFM技术对爱好者和研究人员来说更加易得和负担得起。

## 特性
- **低成本**: 利用3D打印技术降低制造成本。
- **DIY友好**: 设计易于组装和使用。

## 要求
- 3D打印机
- PCB和焊接
- OPU（光学拾取器）
- AFM探针
